## 其他
- 单机使用
    - 优点
        - 效率高，不受网络IO瓶颈限制
    - 缺点
        - 不能保证全局唯一
        - 性能瓶颈在于物理机CPU速度
        - 粗略有序，值不连续，有跳跃
- 单服务使用
    - 优点
        - 全局唯一
    - 缺点
        - 性能瓶颈在于网络IO和物理机cpu速度
        - 服务宕机发号服务即不可用
        - 粗略有序，值不连续，有跳跃
- 分布式环境下使用
    - 优点
        - 高可用
    - 缺点
        - 不能保证全局唯一
        - 性能瓶颈在于网络IO
        - 粗略有序，值不连续，有跳跃
